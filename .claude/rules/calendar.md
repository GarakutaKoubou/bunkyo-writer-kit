# カレンダー登録ルール

## 絶対ルール：公開前はカレンダーに入れない

記事をカレンダーに登録するのは、以下のいずれかの条件が満たされたときのみ：

1. **記事のステータスが `done` になったとき**（Google Driveの完成フォルダへ移動済み）
2. **ユーザーが明示的に「カレンダーに追加して」と指示したとき**

### やってはいけないこと
- `draft` / `review` / `completed` / `writing` / `published` の段階でカレンダーに追加する
- `closed`（公開しなかった記事）をカレンダーに追加・残したままにする
- `hold`（保留中）をカレンダーに追加する

### ステータス変化とカレンダー操作

| ステータス | カレンダー操作 |
|-----------|--------------|
| interview / writing / draft / review | 追加しない |
| completed | 追加しない |
| **done** | **ユーザー指示があれば追加 OK** |
| closed | カレンダーに残っていれば削除する |
| hold | 追加しない |
